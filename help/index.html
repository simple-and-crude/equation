这是个极好的问题！

首先，到现在为止，我没发现任何漏洞。

这个程序分两步生成方程：
1.生成毛坯方程

2.变换方程
1.生成毛坯方程

因为我最喜欢解可以因式分解的方程，所以我决定生成的方程最终能用因式分解法解出来。
例如下面这个：
X2-3X+2=0
因式分解，得
(X-2)(X-1)=0
因为一个数乘0等于0，所以(X-2)和(X-1)中必定有一个是0
可得
X1=2
X2=1
我们可以找到其中的规律。如果二次项的系数a=1，那么一次项的系数b=-X1-X2，常数项c=|X1X2|。所以简陋的毛坯方程应该是长这个样子：
1
X2 + (-X1-X2)X + |X1X2|
 = 
1
0
真是太蠢了，0不能作分母！我们可以随机增加常数项来防止这种事情的发生，其中"%random%"代表一个随机数（不为0，是(x1*101)到(x2*101)之间的数或者其相反数）：
1
X2 + (-X1-X2)X + |X1X2| + %random%
 = 
1
%random%
好！我们现在有了一个简陋的毛坯方程。把变换次数设为0就可以生成这种毛坯方程。


2.变换方程

目前来说，这个程序使用两种方法配合变换方程。
第一种是海星方程变换一式因为太过抽象所以想不出好名字，此方法使用了几个我们幼儿园就知道的定理：
1.方程左右同时乘或除以一个相同的数（0除外），方程仍然成立；

2.分子和分母同时乘或除以一个相同的数（0除外），分数的值不变。
下面请看演示：

请看这个方程，您认为这个方程意味着什么呢？
没错！意味着1/(X2)和1/4是相等的！而且这个方程的解是X=±2。

而且将此方程分成四个象限（左边分子，右边分子，左边分母，右边分母）后，我们发现从其中任选两个象限， 有四种选择（1,2；1,4；2,3；3,4）可以使用上面那两个我们幼儿园就知道的定理，进行乘法或除法变换。

相信您已经发现了，变换时插入的X的位置不同，得到的方程也不同。

以上是两种情况的变换。根据计算，一共有十六种情况（四个象限组合；带X的项的位置；乘或除，4*2*2=16）。观察这十六个情况，可以总结出：
1.方程可以被分成四个象限；

2.四个象限共有4种组合变换方法；

3.当原式除以插入的等式，插入的等式是方程自己时，带X的一项应放在原式中X所在的象限或其对角象限里，否则X会被约掉；

4.当原式乘插入的等式，插入的等式是方程自己时，带X的一项应放在原式中X所在的象限旁边的两个象限里，否则X会被约掉。
这就是“海星方程变换一式”。

相信您茅塞顿开，马上就想到了两秒钟找出原始方程的方法。没想到也没有关系。但我写了这句话就说明我也找到了。我肯定不会让人轻易就解出方程的。
我还有第二种方法：“混沌·神之胡乱构造之术”起名也越来越混沌了可以让方程变得更混乱。
1.通过+-*/四种运算，再来一点随机数，使用“海星方程变换一式”时插入的等式可以变得稍微复杂一点；

2.通过对“稍微复杂了点的等式”使用“海星方程变换一式”和“混沌·神之胡乱构造之术”，可以让等式再复杂一点。
在使用“海星方程变换一式”前，先把插入的等式用“混沌·神之胡乱构造之术”弄得稍微复杂一点，再插入进方程里，就可以让方程更难解一点。
